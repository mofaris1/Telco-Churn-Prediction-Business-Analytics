# Telco Customer Churn Prediction & Business Insights

## Overview
This project aims to address one of the most critical challenges in the telecommunications industry: Customer Churn. By analyzing a dataset of over 7,000 customers, we developed a high-precision machine learning pipeline and a business strategy framework to identify at-risk customers and provide data-driven retention recommendations.

## Key Project Milestones
1. **Data Engineering & Feature Creation:** - Cleaned and preprocessed the Telco dataset.
   - Engineered new features like `Average Monthly Revenue` to gain deeper insights into spending patterns.
2. **Customer Segmentation (Clustering):**
   - Implemented K-Means clustering to segment the customer base into 5 distinct groups, enabling targeted marketing and retention strategies.
3. **Predictive Modeling:**
   - Evaluated multiple architectures (CatBoost, XGBoost, Random Forest).
   - Achieved a **98% test accuracy** with the Random Forest model after handling class imbalance.
4. **Business Intelligence (Dashboarding):**
   - Built an interactive dashboard in Looker Studio to visualize churn drivers, contract types, and segment-specific metrics.

## Strategic Recommendations
The project concludes with a 4-pillar retention strategy:
- Personalized Loyalty Programs for high-value segments.
- Service optimization based on technical support usage.
- Billing process streamlining.
- Competitive pricing and bundled service offerings.

## Technologies Used
- **Python:** Pandas, Scikit-Learn, CatBoost, Imbalanced-learn (SMOTE).
- **Visualization:** Looker Studio, Matplotlib, Seaborn.
- **Domain:** Business Analytics, CRM, Customer Retention.
